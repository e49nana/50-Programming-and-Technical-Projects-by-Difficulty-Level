# 🔁 Technical Unit Converter (Python)

This project is a simple **command-line tool** written in Python that converts between common technical units:

### ✅ Supported Conversions
- Temperature: °C ↔ °F ↔ K
- Pressure: Pa ↔ bar
- Energy: J ↔ kWh
- Power: Coming soon...

### ▶️ How to Use
```bash
python main.py
```

### 🧪 Tests
Run unit tests with:
```bash
python test_converter.py
```
